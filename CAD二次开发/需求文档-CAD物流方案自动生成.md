## 1. 项目背景与目标

### 1.1 业务痛点

作为物流解决方案供应商，在日常业务中存在以下痛点：

- **重复劳动严重**：每个客户方案都需要手动绘制货架、输送机、移动设备、电气柜、安全围栏等，大量时间耗费在重复性绘图操作上
- **调整效率低**：客户需求变更时（如货架尺寸调整、输送路径修改、设备移位），需要逐一修改CAD图元，工作量大且易出错
- **标准化程度不足**：不同设计师出图风格不一致，影响方案的专业性和可读性
- **方案对比困难**：无法快速生成多个备选方案供客户比较

### 1.2 项目目标

开发一款浩辰CAD插件，实现以下核心目标：

1. **参数化生成货架布局**：输入关键尺寸参数，一键生成比例货架平面图/立面图
2. **参数化生成输送机**：输入路径节点和线体参数，自动生成各类输送机布局图
3. **参数化生成移动设备**：快速布置AGV、RGV、堆垛机等移动设备图块
4. **参数化生成电气与安全设备**：自动标注控制柜、触摸屏、安全门、围栏位置
5. **参数化生成辅助设备**：在输送机侧快速布置称重、扫码、外形检测、RFID等模块
6. **提升方案输出效率 70% 以上**：将原本数小时的绘图工作缩短至数分钟
7. **保证出图规范统一**：图层、线型、标注风格标准化

### 1.3 目标用户

- 物流方案设计师
- 售前技术支持人员
- 仓储规划工程师

## 2. 功能需求

### 2.1 比例货架自动生成

#### 2.1.1 功能概述

用户通过面板或命令行输入货架参数，系统自动在CAD当前空间生成符合比例的三视图（俯视图、正视图、侧视图）或指定视图的货架图块。

#### 2.1.2 支持的货架类型

| 编号 | 货架类型 | 说明 | 分期 |
|------|----------|------|------|
| S01 | 横梁式货架 | 最通用的托盘货架，适合大多数仓储场景 | **一期** |
| S02 | 驶入式货架 | 叉车可驶入货架内部存取，适合少品种大批量 | 二期 |
| S03 | 重力式货架 | 利用重力自动滑动，先进先出 | 二期 |
| S04 | 穿梭车货架 | 配合穿梭车使用的高密度存储系统 | 二期 |
| S05 | 中型搁板货架 | 人工拣选，适合小件商品 | 二期 |
| S06 | 悬臂式货架 | 存放长条状物料（管材、板材等） | 二期 |
| S07 | 流利条货架 | 料盒在流利条上滑动，适合拆零拣选 | 二期 |
| S08 | 阁楼式货架 | 多层平台结构，充分利用高度空间 | 二期 |

#### 2.1.3 生成视图

- **俯视图（平面布局图）**：默认生成，用于整体布局规划
- **正视图（立面图）**：可选，展示货架高度方向
- **侧视图**：可选，展示货架深度方向

#### 2.1.4 核心功能点

| 编号 | 功能 | 描述 |
|------|------|------|
| F01 | 单排货架生成 | 输入参数后生成单排货架 |
| F02 | 多排阵列生成 | 输入排数、列数、巷道宽度，自动生成阵列布局 |
| F04 | 尺寸标注 | 自动标注外形尺寸、巷道宽度等关键尺寸 |
| F05 | 货位线绘制 | 在俯视图中自动绘制货位分隔线 |
| F06 | 图层管理 | 自动归类到对应图层（货架外框、货位线、标注等） |
| F07 | 图块定义 | 将单个货架定义为CAD图块，支持重复插入和统一修改 |
| F08 | 参数记忆 | 记住上次使用的参数，减少重复输入 |
| F09 | 预设方案 | 内置常用货架规格预设（如1200×1000标准托盘货架） |
| F10 | 属性文字 | 货架图块附带属性文字（型号、尺寸、承重等） |

### 2.2 输送机自动生成

#### 2.2.1 功能概述

用户指定输送机路径（起点、中间节点、终点）和机型参数，系统自动沿路径生成输送机图元，支持直线段、转弯段、合流/分流节点，并可在输送机上布置辅助设备模块。

#### 2.2.2 支持的输送机类型

| 编号 | 输送机类型 | 说明 |
|------|------------|------|
| C01 | 链条输送机 | 适合重型托盘、大件物料输送 |
| C02 | 滚筒输送机 | 动力/无动力滚筒，适合托盘和重载 |
| C03 | 皮带输送机 | 适合纸箱、散件输送 |
| C04 | 拆码盘机 | 托盘垛拆分成单个托盘，或反向码垛 |
| C05 | 顶升移载机 | 用于输送线交叉节点处的换向移载 |
| C06 | 旋转台 | 用于改变物料输送方向（90°/180°/任意角度） |
| C07 | 提升机/升降机 | 垂直方向输送，实现跨楼层或高差输送 |

#### 2.2.3 核心功能点

| 编号 | 功能 | 描述 |
|------|------|------|
| F11 | 直线段生成 | 指定起点和终点，生成直线输送段 |
| F12 | 多段路径生成 | 多点连线，自动生成含转弯的完整路径 |
| F13 | 转弯段生成 | 在路径转角处自动生成圆弧转弯段（30°/45°/60°/90°） |
| F14 | 合流/分流节点 | 在指定位置生成合流或分流输送节点 |
| F15 | 输送方向箭头 | 自动标注输送方向 |
| F16 | 编号标注 | 输送线编号、分段编号 |
| F17 | 尺寸标注 | 分段长度、总长、转弯半径等 |
| F18 | 图层管理 | 线体轮廓、中心线、标注等分图层 |
| F19 | 标高标注 | 支持标注输送线标高（离地高度） |
| F20 | 设备图例 | 自动生成输送机设备图例表 |
| F21 | 辅助设备挂载 | 在输送机侧自动布置称重/扫码/外形检测/RFID模块 |

### 2.3 移动设备自动生成

#### 2.3.1 功能概述

用户选择移动设备类型并指定参数，系统在CAD中生成对应设备图块（俯视图），支持沿路径布置（AGV/RGV运行路径）和单点布置（堆垛机）。

#### 2.3.2 支持的移动设备类型

| 编号 | 移动设备类型 | 说明 |
|------|--------------|------|
| M01 | AGV（自动导引车） | 潜伏式/背负式/叉车式，适合柔性搬运 |
| M02 | RGV（有轨穿梭车） | 在固定轨道上运行，高速直线搬运 |
| M03 | 堆垛机 | 巷道内沿轨道运行，实现高位存取 |

#### 2.3.3 核心功能点

| 编号 | 功能 | 描述 |
|------|------|------|
| F22 | 单设备图块生成 | 选择设备类型和型号，在指定位置生成俯视图块 |
| F24 | RGV轨道生成 | 沿直线生成RGV轨道双线，标注轨道间距 |
| F25 | 堆垛机巷道生成 | 在货架巷道内自动生成堆垛机轨道和作业范围 |
| F27 | 设备编号标注 | 自动标注设备编号和型号 |
| F28 | 充电/停靠站 | 生成AGV/RGV充电站和停靠点位图块 |
| F29 | 图层管理 | 设备轮廓、运行轨道、标注等分图层 |

| SA03 | 安全光幕 | 光电保护装置，非接触式入口防护 |

#### 2.5.3 核心功能点

| 编号 | 功能 | 描述 |

### 2.7 设备清单自动统计与导出

#### 2.7.1 功能概述

图纸生成完成后，系统自动扫描当前DWG中所有已生成设备的图块，按类型归类统计型号和数量，生成设备清单表格，并支持导出为Excel/CSV格式，直接用于方案报价和采购。

#### 2.7.2 核心功能点

| 编号 | 功能 | 描述 |
|------|------|------|
| F45 | 全图扫描统计 | 扫描当前图纸中所有插件生成的设备图块，按类型归类统计 |
| F46 | 设备清单表生成 | 在CAD图纸中自动生成设备清单表格（含序号、设备类型、型号、数量、备注） |
| F47 | Excel导出 | 将设备清单导出为 .xlsx 文件，支持直接用于报价 |
| F48 | CSV导出 | 将设备清单导出为 .csv 文件，方便数据交换 |
| F49 | 统计范围选择 | 支持全图统计或框选区域统计 |
| F50 | 清单模板配置 | 支持自定义清单表头字段（如增加"品牌"、"单价"等列） |
| F51 | 编号校验 | 自动检测重复编号或缺失编号，输出校验报告 |

| F64 | 删除自动回收 | 删除某设备后，可选择是否自动填补空号 |

## 3. 参数体系设计

### 3.1 货架参数

#### 3.1.1 几何参数（必填）

| 参数代号 | 参数名称 | 单位 | 取值范围 | 说明 |
|----------|----------|------|----------|------|
| L | 货架总长 | mm | 1000 ~ 50000 | 单排货架的总长度 |
| D | 货架深度 | mm | 400 ~ 2000 | 货架进深方向尺寸 |
| H | 货架总高 | mm | 1500 ~ 30000 | 货架地面到顶部的总高度 |
| N_layer | 层数 | 层 | 1 ~ 20 | 横梁层数（含地面层） |
| H_layer | 层高 | mm | 300 ~ 3000 | 每层之间的间距 |

#### 3.1.2 承载参数

| 参数代号 | 参数名称 | 单位 | 取值范围 | 说明 |
|----------|----------|------|----------|------|
| W_layer | 每层承重 | kg | 100 ~ 5000 | 单层最大承载重量 |
| W_total | 总承重 | kg | — | 系统自动计算 = W_layer × N_layer |

#### 3.1.3 布局参数

| 参数代号 | 参数名称 | 单位 | 取值范围 | 说明 |
|----------|----------|------|----------|------|
| N_row | 排数 | 排 | 1 ~ 50 | 货架排列数 |
| N_col | 列数 | 列 | 1 ~ 100 | 每排货架的列数（货位数） |
| W_aisle | 巷道宽度 | mm | 1500 ~ 6000 | 排与排之间的通道宽度 |
| W_pallet | 托盘宽度 | mm | 800 ~ 1400 | 标准托盘宽度 |
| D_pallet | 托盘深度 | mm | 800 ~ 1400 | 标准托盘深度 |
| gap_x | 列间距 | mm | 50 ~ 200 | 相邻货位之间水平间隙 |
| gap_y | 排间距 | mm | 0 ~ 500 | 背靠背货架之间的间隙 |

#### 3.1.4 立柱/结构参数

| 参数代号 | 参数名称 | 单位 | 取值范围 | 说明 |

### 5.5 货架生成核心算法思路

```
输入: 货架参数(P)
输出: CAD图元集合(E)

步骤:
1. 根据 L, D, N_col 计算单个货位宽度 W_cell = (L - gap_x*(N_col-1)) / N_col
2. 根据 D, N_row, W_aisle, gap_y 计算排布间距
3. 俯视图:
   - 绘制每排每列货架矩形外框
   - 绘制立柱位置标记（四角圆圈/方块）
   - 绘制货位分隔线
4. 正视图:
   - 根据 H, N_layer, H_layer 计算横梁位置
   - 绘制立柱竖线 + 横梁横线
5. 侧视图:
   - 根据 D, H 绘制侧面轮廓
   - 标记层板位置
6. 应用图层、线型、颜色
7. 添加尺寸标注和文字标注
```

## 6. 非功能性需求

### 6.1 性能要求

| 指标 | 要求 |
|------|------|

- 参数输入支持实时预览（输入参数后，面板内显示预览草图）

- 提供合理的默认参数值，新用户可直接点击"生成"看到效果
- 错误提示应清晰、中文、描述问题所在

### 6.3 兼容性要求

- 支持 **AutoCAD 2020**（64位）
- 支持 Windows 10 / Windows 11 64位
- DWG文件格式兼容 AutoCAD 2018 及以上

### 6.4 可扩展性要求

- 参数模型支持扩展新的货架/输送线类型
- 图层命名规则可配置
- 标注样式可配置
- 预留BOM导出接口（未来可将生成的图元统计为物料清单）
